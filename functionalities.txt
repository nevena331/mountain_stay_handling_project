Проект: Система за управление на хижи, стаи, туристи, групи и т.н.

Функционалности
Място за спане (SleepingPlace) [Abstract]: Базов клас, който дефинира общите характеристики за всяка точка за нощувка.
    - Полета:
        - string име
        - double географска ширина
        - double географска дължина
        - int височина
Хижа (Hut): Конкретна имплементация на място за спане, която притежава списък от стаи, столова и информация за наличие на ток/вода.
    - Полета:
        - List<Room> стаи
        - bool има столова
        - bool има ток
        - bool има вода
Къмпинг (Camping): Конкретна имплементация на място за спане, фокусирана върху площ за палатки, брой места на открито и такса „триножник“.
    - Полета:
        - Int тент площ (кв.м.)
        - int брой места на открито
        - double такса огън (триножник)
Стая (Room): Управление на леглата в хижите. Пази информация за брой легла, цена на легло и дали стаята е със собствен санитарен възел.
    - Полета:
        - int брой легла
        - double цена на легло
        - bool има собствен санитарен възел
        - string номер на стая
Пътека (Trail): Описва прехода между две точки (SleepingPlaces). Съдържа дължина, денивелация и трудност.
    - Полета:
        - SleepingPlace начална точка
        - SleepingPlace крайна точка
        - double дължина (км)
        - double денивелация (м)
        - int трудност (1-5)

Турист (Tourist): Пази данни за планинаря – име, телефон и списък с вече посетени места.
    - Полета:
        - string име
        - string телефон 
        - List<SleepingPlace> посетени места

Резервация (Booking): Обединява туриста с конкретно място за спане за дадена дата, като изчислява финалната цена.
    - Полета:
        - Tourist турист
        - SleepingPlace място за спане
        - DateTime дата на резервация
        - double финална цена

Група (HikingGroup): Организира няколко туриста в обща единица с водач, което позволява групови резервации и отстъпки.
    - Полета:
        - List<Tourist> туристи
        - Tourist водач
        - double групова отстъпка (процент)
